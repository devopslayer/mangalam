# Frontend Assignment

This project is a responsive webpage built using vanilla HTML, CSS, and JavaScript based on the provided design specifications.

## Features

* Responsive layout for desktop, tablet, and mobile
* Sticky header behavior
* Interactive image section with zoom effect
* FAQ accordion
* Modal forms for brochure download and quote request
* Clean semantic HTML structure

## Tech Stack

* HTML5
* CSS3 (Flexbox, responsive media queries)
* Vanilla JavaScript
